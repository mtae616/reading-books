# Variable

- 변수 : 값을 저장할 수 있는 메모리 상의 공간
- 초기화 : 변수를 사용하기 전에 처음으로 값을 저장하는 것

## 변수의 타입
- 기본형 변수는 실제 값을 저장
  - boolean
  - char
  - byte, short, int, long
  - float, doucle
- 참조형 변수는 주소를 값으로 갖는다. (pointer)
  - object, 위 8개 제외 나머지
- 자바는 참조형 변수 간의 연산을 할 수 없으므로 실제 연산에 사용되는 것은 모두 기본형 변수이다.

### 상수와 리터럴

- final : 상수
- literal : 그 자체로 값 (12, 123, 3.14, 'A')

### 형 변환

- int : 100101100(300) -> byte : 00101100(44)
  - 왼쪽에서부터 짤려 나감
- float -> double
  - 가수부 오른쪽에서 부터 짤려 나감
